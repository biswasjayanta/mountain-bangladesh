# Mountains, Lakes & Waterfalls of Bangladesh

[View Page](https://biswasjayanta.github.io/mountain-bangladesh/)

Interactive 3D terrain viewer built with [Procedural GL JS](https://github.com/felixpalmer/procedural-gl-js).

The Map showing different Mountain peaks, lakes, and waterfalls situated in Chattogram Division, Bangladesh. The datapoint has been collected from [Wikipedia](https://en.wikipedia.org/wiki/List_of_mountains_of_Bangladesh) and the help of [Google Maps](www.maps.google.com).
